# MoviEkopedia

## Description

**MoviEkopedia** is a modern web application for discovering detailed information about movies — including plot, cast, director, genre, and more. It’s part of the **Ekopedia Ecosystem**.

## Tech Stack

- Backend: Laravel 12
- Frontend: VueJS 3
- API: OMDb API
