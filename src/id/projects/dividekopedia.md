# DividEkopedia

## Deskripsi

**DividEkopedia** adalah aplikasi web modern untuk membagi tagihan dengan banyak orang. Ini adalah bagian dari **Ekosistem Ekopedia**.

## Tech Stack

- Backend: Laravel 12
- Frontend: VueJS 3
