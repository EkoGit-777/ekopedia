# AideEkopedia

## Deskripsi

**AideEkopedia** adalah aplikasi obrolan bertenaga Kecerdasan Buatan. Ini adalah bagian dari **Ekosistem Ekopedia**.

## Tech Stack

- Backend: Laravel 12
- Frontend: VueJS 3
- API: gemini-2.5-flash
