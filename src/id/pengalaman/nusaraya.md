# PT Nusantara Baskara Jaya

## Tentang Perusahaan

Nusaraya (Nusantara Baskara Jaya) adalah perusahaan pegembang bidang IT, mereka senang mengatasi masalah kelas dunia dengan pendekatan unik mereka dalam visi komputer, teknologi seluler, media, sistem cloud, telekomunikasi, dan layanan publik.

## Linimasa

:calendar: Jan 2016 - Mei 2020

## Kontribusi Saya

- Shafira
Ini proyek pertama saya di sana. Saya membangun aplikasi seluler berbasis web menggunakan AngularJS dan PhoneGap.
- Aji Mumpung
Aji Mumpung adalah aplikasi seluler berbasis hadiah. Saya membantu mengembangkan layar kunci untuk aplikasi tersebut dan menangani masalah pengguna. Aplikasi ini dibangun menggunakan Java sebagai frontend Android native dan Laravel sebagai backend.
- Eilhamzah
Eilhamzah adalah pengelola sistem paket umroh. Paket ini dibangun menggunakan kerangka kerja Yii. Saya membantu melakukan refaktor fitur keuangan dan juga mengintegrasikan laporan keuangan dengan Zahir.

## Perjalanan Saya

Sebagai karier pertama saya di bidang TI, pengetahuan dan pengalaman saya terbatas. Pengetahuan yang saya miliki sebagian besar adalah bahasa pemrograman untuk aplikasi desktop seperti C++, Java, dan Visual Basic. Saya hampir tidak memiliki pengetahuan tentang pengembangan web. Ketika mengerjakan proyek pertama saya, saya belajar HTML dan JavaScript selama sekitar 2 minggu. Sebagian besar riwayat penelusuran saya berasal dari W3Schools. :sweat_smile:

Setelah saya memahami HTML dan JavaScript, saya tidak butuh waktu lama untuk meningkatkan kecepatan kerja saya. Ketika saya dipindahkan ke Eilhamzah, itu menjadi tantangan tersendiri bagi saya. Untungnya, saya telah belajar sedikit tentang PHP ketika saya berada di tim Aji Mumpung. Meskipun saya tidak lama menangani backend, saya sudah mengerti intinya. Dan di Eilhamzah, saya belajar lebih banyak tentang konsep MVC.
