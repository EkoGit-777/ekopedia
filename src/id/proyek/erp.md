# ERP ringan untuk ritel

## Deskripsi

Salah satu proyek terbesar yang saya kerjakan di Gradin. ERP ringan untuk ritel dengan banyak cabang di Surabaya. ERP ini memiliki fitur untuk pergudangan, pembelian, penyimpanan stok, penjualan ritel, dan penjualan online.

## Tech Stack

- Backend: Laravel 10
- Frontend: VueJS 3
