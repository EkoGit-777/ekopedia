# Hari Pertama Bekerja Jarak Jauh

Hari ini adalah hari pertama saya bekerja jarak jauh sebagai Software Engineer. Saya menggunakan Ruby on Rails pada proyek yang sedang saya kerjakan. Karena saya belum pernah menggunakan Ruby on Rails sebelumnya, kecuali saat melakukan pengujian untuk melamar pekerjaan ini, saya harus menghadapi beberapa tantangan. Tugas saya adalah membiasakan diri dengan proyek dan juga memperbarui pengujian unit dan dokumentasinya. Sebagai pengembang yang telah menggunakan Laravel selama beberapa tahun, saya tidak mengalami kesulitan memahami struktur dan beberapa konsep yang digunakan dalam Ruby on Rails. Namun, ada beberapa perbedaan yang sering membuat saya salah paham tentang kode tersebut. Misalnya, di Laravel Factory, kuncinya selalu berupa nama kolom tabel. Namun, di Ruby on Rails, ternyata tidak selalu demikian. :sweat_smile:

Meskipun tidak semulus yang saya harapkan, saya dapat mengatasinya dan berhasil memperbaiki beberapa pengujian unit. Berkat chatgpt, saya dapat lebih memahami Ruby on Rails. Chatgpt bahkan memberi saya peta mental untuk migrasi dari Laravel ke Ruby on Rails. :star_struck:
Kamu bisa cek di [sini](/posts/laravel-rails.html) kalau mau lihat-lihat. :wink:
